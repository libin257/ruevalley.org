### 工具和脚本
1. **`generate-mdx-content.py`** - Python 内容生成脚本

   - 智能识别内容类型
   - 生成针对性内容模板
   - 完整的 SEO 元数据

2. **`scrape-and-fill-mdx.js`** - Node.js 模板脚本 (备用)

## 使用方法

### 重新生成所有内容
```bash
python3 scripts/generate-mdx-content.py
```

### 更新特定内容
1. 编辑对应的 MDX 文件
2. 或修改 CSV 后重新运行脚本

### 添加新内容
1. 在 CSV 中添加新行
2. 运行生成脚本
3. 新文件将自动创建
